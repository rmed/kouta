Detailed Description

template<class TWrapped>
class kouta::base::Worker< TWrapped >

Background component executor.

A Worker wraps a Component based object so that the event loop it is attached to is executed in a separate thread. This implies that any children that have the wrapped Component as a parent will also run in the worker thread.

By default, the Worker does nothing, and its event loop must be explicitly started by calling the run() method. As opposed to the original method defined in Root, the one specified here will launch the thread (and the event loop) and return immediately.
